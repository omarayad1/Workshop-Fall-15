# Introduction to CSS

--

### Basic styling

here we are giving styles to every link element on the page

```css
a {
	color: #eee;
	font-size: 12em;
	margin-top: 10%;
}
```

---

### Selectors

selectors are used in css to give style to specific object in the html page

---
- referencing IDs

```css
#stuff {
	border: 1em;
	background-color: red;
	margin-left: 20%;
}
```
---
- referencing Classes

```css
.stuff {
	border: 1em;
	background-color: red;
	margin-left: 20%;
}
```
